The application must include the following modules.

Dashboard
- Show certifications completed
- Show next scheduled exam
- Show resume status
- Show LinkedIn status
- Show training amendment status

Certification Tracking
Each certification record must include:
- name
- vendor
- status
- exam_date
- exam_score
- completion_date
- plan_type

Resume Tracking
- upload current resume
- track resume status

LinkedIn Tracking
- store profile URL
- track profile status

Training Plan Amendments
Each amendment must include:
- title
- description
- status
- created date

Documents
- allow file upload
- associate files with certifications or amendments
